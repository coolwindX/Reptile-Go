# Reptile-Go
某科学的多功能爬虫脚本

## 介绍
随着信息爆炸时代的到来,高效地从网络上获取信息成为一个难题,但是`Reptile-Go`可以帮到你,本项目旨在为多种情况下的互联网信息进行聚合

本项目仅供学习交流,请勿进行不法行为和商业用途

最近抽奖中了100万,所以准备好好享受一下生活，暂时鸽一下，会有的会有的
## 功能
- [ ] 搜索引擎文本内容聚合
- [ ] 自定义链接网页爬取
- [ ] 图片爬取
- [ ] 网页提交请求
- [ ] 试题爬取(向某位不知名的学习猛男致敬)
